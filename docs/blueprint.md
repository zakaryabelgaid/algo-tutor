# **App Name**: C-Tutor Pro

## Core Features:

- Interactive C Tutorials: Display structured C tutorials, mirroring the W3Schools format, fetched from the Firestore database. The tutorials include examples and exercises.
- Teacher Registration & Approval: Teachers can register. Admins must approve them before they gain full access, managed via Firestore.
- Q&A System: Visitors can ask questions. These are stored in Firestore and emailed to teachers.
- Integrated C Compiler: Integrate Programiz's client-side compiler for testing code snippets directly within the platform.
- Grade/Year Assignment: Teachers can assign a grade/year to lessons and announcements, which will be saved to Firestore.
- Personalized Learning Recommendations: AI tool to analyzes user progress and recommends relevant tutorials, exercises, and examples from the Firestore database to improve their understanding of C programming.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to represent technology and knowledge.
- Background color: Light grayish-blue (#E6F7FF) for a clean, modern look.
- Accent color: A complementary orange (#FF8C00) to highlight key interactive elements and calls to action.
- Body and headline font: 'Inter', a sans-serif font known for its clean and readable design.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use a set of modern, flat icons to represent different sections and functionalities. Icons should be simple and easily recognizable.
- Incorporate subtle animations, such as transitions and loading spinners, to enhance user experience and create a dynamic feel.
